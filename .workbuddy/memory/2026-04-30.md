   - 将按钮容器从 QLayout 改为 QWidget + QVBoxLayout 包装
   - 容器设置 QSizePolicy::Expanding + QSizePolicy::Fixed，确保高度只由内容决定
   - 按钮容器 layout 设置 Qt::AlignTop
   - rootLayout 对容器使用 `addWidget(navContainer, 0, Qt::AlignTop)` 而非 stretch
   - 移除 rootLayout 的 setAlignment(Qt::AlignTop)（QLayout 不支持这个）
